# Cloud-Based File Storage System

A full-stack cloud application for securely uploading, managing, downloading, and sharing files using Microsoft Azure Blob Storage.

## 🚀 Tech Stack

### Frontend
- React.js
- React Router
- Tailwind CSS
- Vite

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Cloud
- Azure Blob Storage

### Authentication
- JWT
- bcrypt

### Deployment
- Azure Static Web Apps
- Azure App Service
